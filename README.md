# Zepto-SQL-DA
This project is a step-by-step Data Analytics Portfolio Project built using SQL and MySQL on a real-world e-commerce dataset from Zepto. The project demonstrates how a data analyst works with raw retail inventory data, from database creation and data cleaning to exploratory analysis and business insights generation using SQL. This project is designed to be showcasing practical SQL and analytical skills used in real-world business environments.

Dataset:
The dataset contains e-commerce inventory and product-related information from Zepto’s quick-commerce platform.

Dataset Includes:
1. Product names
2. Categories
3. Pricing information
4. Stock availability
5. Inventory details

Key Challenges:
1. Missing values
2. Duplicate records
3. Zero or invalid pricing
4. Unstructured retail data

Tools:
1. MySQL

Project Steps:
1. Database Setup
- Imported raw CSV data into MySQL.
- Created tables and structured the database.

2. Data Exploration - Performed exploratory analysis using SQL
- Checked missing values
- Identified duplicates
- Explored product categories
- Understood dataset structure

3. Data Cleaning
- Handled zero pricing issues
- Converted paise to rupees
- Removed inconsistent records
- Standardized formats
 
4. Exploratory Data Analysis (EDA)
- Analyzed:
a) Product distribution
b) Category trends
c) Pricing insights
d) Inventory patterns

5. Business Analysis- Solved real-world business problems
- Revenue analysis
- Stock analysis
- Product segmentation
- High-performing categories
- Inventory insights

Dashboard: 
- The insights generated from SQL analysis can be further visualized using tools like- Power BI, Tableau or Excel.
- Possible dashboard metrics:
a)Total revenue
b)Top-selling categories
c)Inventory status
d)Product pricing trends
e)Stock distribution

Results- helped in understanding:
- Understanding real-world retail datasets
- Improving SQL query writing skills
- Learning practical data cleaning techniques
- Performing business-focused analysis
- Building a portfolio-ready analytics project
