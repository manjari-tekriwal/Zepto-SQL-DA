# Zepto-SQL-DA
This project is a step-by-step Data Analytics Portfolio Project built using SQL and PostgreSQL on a real-world e-commerce dataset from Zepto. The project demonstrates how a data analyst works with raw retail inventory data — from database creation and data cleaning to exploratory analysis and business insights generation using SQL. This project is designed to be portfolio-ready and interview-friendly, showcasing practical SQL and analytical skills used in real-world business environments.

Dataset:
The dataset contains e-commerce inventory and product-related information from Zepto’s quick-commerce platform.

Dataset Includes:
Product names
Categories
Pricing information
Stock availability
Inventory details
Key Challenges:
Missing values
Duplicate records
Zero or invalid pricing
Unstructured retail data

Tools & Technologies
MySQL
SQL

Project Steps
1. Database Setup
Imported raw CSV data into PostgreSQL
Created tables and structured the database
2. Data Exploration

Performed exploratory analysis using SQL:

Checked missing values
Identified duplicates
Explored product categories
Understood dataset structure
3. Data Cleaning

Cleaned and transformed the data:

Handled zero pricing issues
Converted paise to rupees
Removed inconsistent records
Standardized formats
4. Exploratory Data Analysis (EDA)

Analyzed:

Product distribution
Category trends
Pricing insights
Inventory patterns
5. Business Analysis

Solved real-world business problems using SQL:

Revenue analysis
Stock analysis
Product segmentation
High-performing categories
Inventory insights
